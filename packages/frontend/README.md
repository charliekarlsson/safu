# Frontend (SAFU Auth)

Vite + React + TypeScript landing and demo UI for the Solana micro-transaction auth flow.

## Run

```bash
pnpm install
pnpm dev:frontend
```

Dev server proxies `/api` to the backend on `http://localhost:8787`.

## Sections
- Hero explaining the protocol
- Code snippet cards (API surface, listener sketch)
- Flow timeline and live demo card (shows recipient + amount, polls auth status)

Styling uses a dark palette with pixel monospace accents (Press Start 2P + Sora).
